Markdown syntax

发表于 2017-06-25

Markdown syntax – It will help you write your blog easily

Title

1. Use = and - to define title

  • The most important title
    =======================
  • The second important title
    -----------------------

2. Use # to define title

  • # The most important title
  • ## The second important title
  • ### The third important title
  • #### The forth important title
  • ##### The fifth important title
  • ###### The sixth important title

Paragraph

Use space line before and after a paragrah. If you want to force to enter a newline in a paragraph.
You can use more than 2 spaces and a enter.

Block

Before the first line or every line of paragraph, use > to define a block. Also you can use >> to define a block in the first block. Like this:

The first block

The second block

Code Block

Use 4 spaces or 1 tab before every line in a paragraph. Also need a space line before and after the code block.
For example: 

void main()
{
    return null;
}

Emphasis

  • Use *text* to make the text italic, *around the text.
  • Use **text** to make the text bold, **around the text.

List

1. unordered lists

Use - or + or * to define the out of order,like this:

  • The first one
  • The second one
  • The third one

2. ordered list

Use 1. 2. to define the orderd list,like this:

  1. The first one
  2. The second one
  3. The third one
